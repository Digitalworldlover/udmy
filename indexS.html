CSS selectors are used to "find" (or select) the HTML elements you want to style. Learn from over 300 examples! With our editor, you can edit the CSS, and click on a button to view the result.

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.
